# 🔄 Obsidian MultiSync

> **Sync your Obsidian vault across multiple cloud drives — simultaneously.**

⚠️ **Disclaimer:** This plugin modifies files in your vault and cloud storage. **Always maintain offline backups.** The author is not responsible for any data loss. Current safeguards include sync confirmation dialogs, cloud trash bins, and local file trash — but no guarantee is absolute.

---

## ☁️ Multi-Cloud Sync

Sync your vault with **multiple cloud storage providers at the same time**.

Currently supported:
- **Dropbox**
- **Google Drive**
- **OneDrive**

More providers coming soon.

## 📁 Easy Migration

Migrating files between cloud drives is as simple as **drag and drop**. Move files from one mapped folder to another in Obsidian's file explorer — the next sync uploads to the new drive and cleans up the old one.

![File Explorer](img2.png)

---

## 🛡️ Safe Deletion

Every delete operation goes through multiple safety layers:

1. **Confirmation dialog** — review all changes before they happen
2. **Cloud trash bin** — deleted cloud files are recoverable from the provider's trash
3. **Local trash** — deleted local files go to Obsidian's `.trash` folder
